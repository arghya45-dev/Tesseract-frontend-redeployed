# 🌟 Overview

TESSERACT 2K26 is a modern, high-performance web platform designed to serve as the digital face of one of the largest technical festivals organized by GNIT.

The platform brings together participants, organizers, sponsors, and technology enthusiasts into a single interactive ecosystem where users can:

✅ Explore events

✅ Register for competitions

✅ Discover workshops

✅ View merchandise

✅ Track festival countdowns

✅ Connect with organizers

✅ Experience the energy of the fest through a futuristic interface

Rather than being a simple event website, TESSERACT is designed as a complete digital engagement platform that enhances participant experience before, during, and after the event.

---

# 🎯 Problem Statement

Managing large-scale college technical festivals often involves:

- Scattered information
- Manual registrations
- Poor user experience
- Lack of participant engagement
- Difficult event discovery

TESSERACT solves these challenges through a centralized, responsive, and visually engaging platform that streamlines event discovery and participation.

# ⚙️ Local Setup

## Clone Repository

```bash
git clone https://github.com/arghya45-dev/Tesseract.git
```

## Enter Project Directory

```bash
cd Tesseract
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Application runs at:

```text
http://localhost:5173
```
